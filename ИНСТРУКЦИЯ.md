# Пошаговая инструкция по запуску агента

## Шаг 1: Откройте Терминал

**На macOS:**
- Нажмите `Cmd + Space` (Command + пробел)
- Введите "Terminal" и нажмите Enter
- Или откройте через: Finder → Программы → Утилиты → Терминал

## Шаг 2: Перейдите в папку проекта

В открывшемся терминале введите и нажмите Enter:

```bash
cd /Users/pavelkokora/crewai_agent
```

## Шаг 3: Установите зависимости

**ВАЖНО:** Рекомендуется использовать виртуальное окружение Python для изоляции зависимостей.

### Вариант A: С виртуальным окружением (рекомендуется)

```bash
# Создайте виртуальное окружение
python3 -m venv venv

# Активируйте виртуальное окружение
source venv/bin/activate

# Установите зависимости
pip install -r requirements.txt
```

После установки, когда будете запускать агента, не забудьте активировать виртуальное окружение:
```bash
source venv/bin/activate
python3 main.py
```

### Вариант B: Без виртуального окружения

Попробуйте одну из команд (по очереди, какая сработает):

```bash
pip3 install --user -r requirements.txt
```

или

```bash
python3 -m pip install --user -r requirements.txt
```

или (требует sudo, менее безопасно)

```bash
sudo pip3 install -r requirements.txt
```

## Шаг 4: Создайте файл .env с API ключом

### Вариант A: Через терминал (если у вас есть API ключ Serper.dev)

```bash
echo "SERPER_API_KEY=ваш_ключ_здесь" > .env
```

Замените `ваш_ключ_здесь` на ваш реальный API ключ от Serper.dev

### Вариант B: Через текстовый редактор

1. Откройте папку проекта в Finder:
   ```bash
   open /Users/pavelkokora/crewai_agent
   ```

2. Создайте новый файл с именем `.env` (обратите внимание на точку в начале!)

3. Откройте файл `.env` в любом текстовом редакторе и добавьте:
   ```
   SERPER_API_KEY=ваш_ключ_здесь
   ```

4. Сохраните файл

**Где взять API ключ Serper.dev?**
- Зайдите на https://serper.dev/
- Зарегистрируйтесь (или войдите, если уже зарегистрированы)
- Скопируйте ваш API ключ из личного кабинета
- Вставьте его в файл `.env`

## Шаг 5: Запустите агента

В терминале выполните:

```bash
python3 main.py
```

или

```bash
python main.py
```

## Что произойдет?

Агент:
1. Найдет последние новости про AI Agents в интернете
2. Напишет блог-пост на русском языке
3. Сохранит результат в файл `blog_post.txt`

## Если что-то пошло не так

**Ошибка "No module named 'crewai'"**
→ Выполните установку зависимостей снова (Шаг 3)

**Ошибка "SERPER_API_KEY not found"**
→ Проверьте, что файл `.env` создан правильно и содержит ваш API ключ

**Ошибка при поиске в интернете**
→ Проверьте, что ваш API ключ Serper.dev действителен и у вас есть доступ к интернету
